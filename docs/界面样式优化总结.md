# 界面样式优化总结

```css

--primary-light: #E9D5FF;
```

### 5. 样式分离原则

✅ **关注点分离**
- 每个页面有独立的CSS文件
- 全局样式统一在 `app.css` 中管理
- 组件样式统一在各自的CSS文件中

✅ **可维护性**
- 使用CSS变量管理颜色主题
- 使用语义化的类名
- 避免内联样式

✅ **可复用性**
- 统一的组件样式覆盖
- 统一的字体和圆角规范
- 统一的颜色系统

✅ **响应式设计**
- 所有页面支持移动端适配
- 侧边栏在移动端自动隐藏
- 表格和卡片自适应宽度

## 使用方法

### 在新页面中使用样式

1. 创建页面专用CSS文件：
```css
/* /css/your-page.css */
.your-page {
    /* 页面容器样式 */
}
```

2. 在Razor组件中引用：
```razor
<link href="/css/your-page.css" rel="stylesheet" />

<div class="your-page">
    <!-- 页面内容 -->
</div>
```

3. 使用全局CSS变量：
```css
.your-component {
    color: var(--text-primary);
    background-color: var(--bg-card);
    border: 1px solid var(--border-color);
}
```
